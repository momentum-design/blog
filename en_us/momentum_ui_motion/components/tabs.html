mm-tabs

usage

<mm-tabs id='myTab' current='1'></mm-tabs>

Attribute

  • current

    Set the selected tab.

Method

  • select

    To set the selected tab.

document.getElementById('myTab').select(0);

Properties

  • Events

    Bind evnets to tabs.

document.getElementById('myTab').Events.bind('onSelect', (index) => {} );
